# easystats
A statistics library for Python written in C during the VT Hacks X hackathon.

### By: Patrick and Grady  

# Installation  
1. change into `easystats` directory  
2. run `make`  

# Usage  
Just import into your python project  
```
import easystats
print(easystats.mean([1.5,2.64,3.31]))
```
